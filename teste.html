Benefícios do uso das tags semânticas do HTML

Você sabe para que serve as semânticas HTML?

A semântica HTML é muito importante, pois ajuda na compreensão do código, quando utilizado de forma correta, ó código fica fácil de um desenvolvedor entender,
chamamos isso de boas práticas da programação. Então quando utilizado em grandes projetos, trará benefícios para a equipe toda.

Tags HTML básicas

  • <!DOCTYPE>: define o tipo do documento
    • 
                
  • <html>: define o documento em formato HTML
    • 
                
  • <head>: contém metadados/informações sobre o documento
    • 
                
  • <title>: define o titulo do documento, irá aparecer na janela do navegador, não aparece no corpo do documento
    • 
                
  • <body>: define o corpo do documento, o que de fato aparece na página
    • 
                
  • <h1 até h6>: define o titulo do documento, onde o numero define o tamanho, no mínimo 1, no máximo 6
    • 
                
  • <p>: define um paragráfo
    • 
                
  • <br>: coloca uma quebra de linha
    • 
                
  • <hr>: coloca uma superlinha na página
